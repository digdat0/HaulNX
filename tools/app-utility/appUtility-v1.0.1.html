HaulNX  App Utility

Send to Switch ·  both devices on the same network
Shown on the Switch as IP:PORT/CODE. The IP and port are remembered; the code changes each time you open the receive screen.
Sends the collection you're editing here. On the Switch, open Settings › Manage data › Import collection. It shows the address to type above. Leave that screen open — it stops listening once a file arrives, and you confirm the import there.
Pushes a freshly built HaulNX .nro to the console — no USB cable needed, and the same version as installed is fine. On the Switch, open Settings › Check for updates › Over Wi‑Fi; you confirm the install there and restart to switch to the new build.
